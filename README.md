List of what is implemented in this package:
1. Full modular character models.
2. Modular character system C++.
3. Expanded Character Attributes System:
  - Healh, Stamina, Bleeding, Toxic Damage, RadToxic Damage, PsyToxic Damage, Weight, Food, Water, Cold Damage, Poisoning Damage, Fire Damage, Electric Damage, Physical Damage.
  - Creating basic character characteristics.
  - Creating secondary characteristics for a character.
  - Creating a StatsBar for Basic Stats
  - Creating a Condition for characteristics.
  - Creating a logic of good and bad stats for a character.
4. AnimStat and WeaponStat for Player.
5. Expanded ItemTable and ItemTypes via Primary Data Assets.
6. Spatial Inventory aka Stalker2.
7. Equipment Widget aka Stalker2.
